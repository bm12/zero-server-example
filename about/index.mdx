# About ZeroServer

**Zero** is a web framework to simplify modern web development.

It allows you to build your application without worrying about package management or routing.

It's as simple as writing your code in a mix of [Node.js](https://github.com/remoteinterview/zero/tree/master/docs/nodejs/README.md), [React](https://github.com/remoteinterview/zero/tree/master/docs/react/README.md), [HTML](https://github.com/remoteinterview/zero/tree/master/docs/html/README.md), [MDX](https://github.com/remoteinterview/zero/tree/master/docs/mdx/README.md), and [static files](https://github.com/remoteinterview/zero/tree/master/docs/static/README.md) and putting them all in a folder.
